AMRD/PAMR Newtonian Boson Star (nbs) example code



PAMR (Parallel Adaptive Mesh Refinement) features:
  • manages distributed grid hierarchies of the type we tend to use for finite-difference based solution of field equations
  • functions to synchronize/interpolate/inject grid functions within a hierachy
  • internally stores the relevant data structures, and provides a complete set of functions to set/query the relevant structures
AMRD (Adaptive Mesh Refinement Driver) features:
  • implements a Berger and Oliger AMR driver (with modifications to incorporate elliptic equations, and hence provides a FAS multigrid (MG) algorithm)
  • uses PAMR to handle parallel support and data-structure management
  • problem specific variables/grid-parameters/etc. specified using a text parameter file (using RNPL-style variables)
  • numeric algorithm is hard-coded; problem specific code is called at fixed locations within the algorithm
